How many amino acids would the following DNA base sequence make? GGGTTTAAACCCTTTGGGTTTAAA

(A) 3

(B) 4

(C) 6

(D) ** 8

(E) 24

EXPLANATIONS BELOW

Concept note-1: -Because there are only 20 different amino acids but 64 possible codons, most amino acids are indicated by more than one codon. (Note, however, that each codon represents only one amino acid or stop codon.)

Concept note-2: -Expert-Verified Answer. Answer: Four amino acids can be coded for by this message.

Concept note-3: -Use the codon table to determine the sequence of amino acids that will be produced from a DNA or mRNA sequence. Use the codon table to determine which amino acid will be carried by a tRNA, given the anticodon sequence. Identify untranslated regions and include these regions in gene diagrams.

Concept note-4: -Each codon corresponds to a single amino acid (or stop signal), and the full set of codons is called the genetic code. The genetic code includes 64 possible permutations, or combinations, of three-letter nucleotide sequences that can be made from the four nucleotides.
