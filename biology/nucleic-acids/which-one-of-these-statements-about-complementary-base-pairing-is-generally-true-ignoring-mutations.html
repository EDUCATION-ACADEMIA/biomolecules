Which one of these statements about complementary base pairing is generally true (ignoring mutations)

(A) Adenine only ever binds to thymine

(B) Any purine can bind with any pyrimidine

(C) ** Cytosine only binds to guanine

(D) ** Adenine can only bind with thymine or uracil

(E) ** Complementary bases pair through hydrogen bonding

EXPLANATIONS BELOW

Concept note-1: -Answer and Explanation: The correct statement is a because DNA complementary base pairing occurs when C binds to G and A binds to T. Statement b is incorrect because adenine base pairs with thymine, not guanine, and cytosine base pairs with guanine, not thymine.

Concept note-2: -Within a molecule of DNA, pairs of bases are chemically bonded together by hydrogen bonds. These bonds hold the two strands of the molecule together. Bases pair up with each other in a consistent way, and this process is called complementary base pairing, or Watson-Crick base pairing.

Concept note-3: -A DNA adenine to an RNA thymine complementary base pairing does not exist because it is RNA, The base thymine is not present: Instead of thymine uracil is present which consists of a very similar structure like thymine.

Concept note-4: -Answer and Explanation: The correct answer: The given option which is not correctly matched with its complementary base pair for either DNA or RNA is (d) Uracil-Cytosine.
