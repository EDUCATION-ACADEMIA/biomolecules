NUCLEIC ACIDS

The DNA sequence ATCAGCGCTGGC is part of a gene. How many amino acids are coded for by this message?

(A) ** 4

(B) 8

(C) 12

(D) 20

EXPLANATIONS BELOW

Concept note-1: -Answer: Four amino acids can be coded for by this message.

Concept note-2: -In all living things, the DNA is able to code for 20 amino acids. These amino acids are used to build all of the proteins in every living thing on Earth.

Concept note-3: -There are 64 different codons: 61 specify amino acids and 3 are used as stop signals.

Concept note-4: -The linear sequence of amino acids within a protein is considered the primary structure of the protein. Proteins are built from a set of only twenty amino acids, each of which has a unique side chain. The side chains of amino acids have different chemistries.
